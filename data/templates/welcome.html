Spock Wiki

Spock is a simple wiki software heavily inspired by GitHub's Gollum and is mostly intended as a personal documentation system.

Use with caution, it's still in it's early stages.

Get started

Markup supported

Spock supports:

  • Markdown with GitHub code blocks
  • RestructuredText through rst2html

git

The underlying storage is a git repository.

Other features

Spock does not support many of the features that you will find in a typical Wiki; for example there is not concept of linked pages, as each page is just a text file inside a directory.
